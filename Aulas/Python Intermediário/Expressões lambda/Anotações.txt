-------------->>> Lambda <<<--------------
Lambda, também conhecido como "função anônima" (por não receber um nome) é uma função criada dentro de uma variável.
Exemplo:

a = lambda x,y: x * y
print(a(2,2))

Retorna:
4

Isso é exatamente a mesmo coisa que isso:

def funcao(arg, arg2):
    return arg * arg2

var = funcao(2,2)
print(var)

Só que uma forma bem mais simplificada e agradavel
